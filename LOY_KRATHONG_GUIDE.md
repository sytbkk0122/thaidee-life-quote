## 🏮 水燈節是什麼？

水燈節 (Loy Krathong) 是泰國最重要且最美麗的傳統節日，距今已有 800 多年歷史。

### 節日意義
- **Loy** = 漂浮
- **Krathong** = 燈籠/蓮花燈

**目的**: 
1. ✨ 感謝水神的眷顧
2. 🙏 祈求財運和幸福
3. 🕯️ 放走厄運和煩惱
4. 💑 象徵愛情和團圓

### 傳統習俗
```
放水燈 → 點蠟燭 → 獻花和硬幣 → 祈福願望 → 放入河中
```

人們將燈籠放入河中，象徵將過去的不快洗淨，帶來新的希望。
